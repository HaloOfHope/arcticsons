Sheet Set-Up & Use

I'll be going over the very basics to help you get up and running - if you have more questions about how sheets work in general, google is your friend.

Section I
Sheet Pages

Options

This allows you to change a few things, such as the filler MYO image and the ID prefix, to better fit your species and taste. Notes are added to the sheet for clarity, but you can remove all the text from them to delete them if they're getting in the way.

  • Prefix for Autonumber
    The default letters that preceed the ID number. I suggest no more than 4.
  • Number System
    The initial amount of numbers you want shown in the ID.
    0 = CHA1
    00 = CHA01
    000 = CHA001
    and so on and so forth.
  • MYO Placeholder Image

    The default image you want shown if there isn't a design yet.

The following will be used to fill out the log dropdowns (a QOL feature for logging) and to add species specific prefixes or MYO images.

  • Design Types
    List of design types. (I use this to separate MYOs, Guests Designs, and Official Designs).
  • Statuses
    List of trade statuses.
  • Transaction Types
    List of transactions types.
  • Species
    List of species.
  • Prefixes
    List of prefixes for the autonumber that will be used based on the species of the entry. Once you fill this out, the autonumber will stop using the default prefix completely for all entries.
  • MYO Placeholder
    List of MYO images that will be used based on the species of the entry.

Masterlist Log

This is the list of columns in the Masterlist Log sheet. If a column is optional, that means you can remove it without breaking charadex.

  • [Required] ID

    This automatically adds a transaction number to your new entry. Do not edit this at all. If you add your own ID number, it will break the formula.

    Note: The number will not reset when you add a new entry with a new species. This is a limitation of Google Sheets.

  • [Required] Update ID
    If your updating an old entry, fill this in with said entry's ID.
  • [Optional] Image

    The image for said entry. This column has a special formula - only delete this column if your dead set on never using images.

    If you don't want to use it but don't want to delete it, highlight the entire image column, right click, and select 'hide column'.

  • [Optional] Species
    The dropdown will fill with the list of species you made in Options. If you've also filled out the prefixes list, it will change the number prefix to the one you assigned for the species.
  • [Optional] Owner
    Owner of design at time of logging.
  • [Optional] Artist
    Artist of image at time of logging.
  • [Optional] Designer
    Designer of design at time of logging.
  • [Optional] Worth
    Worth of design at time of logging.
  • [Optional] Status
    Trade status of design at time of logging. The dropdown will fill with the list of statuses you made in Options.
  • [Optional] Design Type
    Type of design. The dropdown will fill with the list of design types you made in Options.
  • [Optional] Transaction Type
    If it's a new entry, traded, etc. The dropdown will fill with the list of transaction types you made in Options.
  • [Optional] Notes
    Any extra notes about the design. These will be overwritten if you update them via a new line so stay frosty!
  • [Optional] Proof
    Proof of transaction.

Public Masterlist

This is the final conglomerate of the logs, and the sheet you'll be pulling information from.

You'll only need to edit this sheet if you're adding extra columns.

Warnings

  • Original log entries should not be deleted - it will shift the ID numbers.
  • I do not recommend logging URLs anywhere other than the Image column.
  • Adding more images will require knowledge of JS and List.js to show them on site.

Section II
Manipulating Sheet Data

Adding an Entry

Step One: Right click on row 2 and then click Insert 1 Above.

Step Two: Add information. If it's a new entry, leave Updated ID blank. If you're updating, make sure to write out the ID.

Step Three: You're done! But make sure to check the bank once you're done in case something doesn't look right.

Changing Usernames

Step One: In the log, press CTRL + F and then type in the old username.

Step Two: Press the button with 3 dots and enter the Replace with portion with the new username.

Step Three: Press Replace all and you're done!

Adding Traits (to the Sheet Only)

Step One: Make the trait row in the Log table, I recommend putting it after Designer.

Step Two: Make the same rows in the Masterlist table.

Step Three: Click any of the headers that are not ID or Image. Press ctrl + c to copy, THIS IS IMPORTANT. Do not just copy the formula from the formula bar.

Step Four: Click on any of the new trait row headers and press ctrl + v. This will paste the formula and automatically change the referenced cells so you don't have to change anything in the formula.
