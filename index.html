About

After Eclipse took out DeviantArt, I made Charadex to help other species creators organize and keep easier track of their masterlist.

And it's not limited to species! If you just sell adopts or what to make a little organizer for your characters - this might help. I use it to keep track of role play AU's and my own designs too!

Please note I'm not an actual dev and have absolutely no formal training. I've just been learning JS here and there and I feel like even jank code is better than no code LOL

Getting Started

I will only be showing how to host on Neocities. You can host anywhere that takes .html, .css, and .js files but I feel like this is the easiest for everyone.

  • Copy this Google Sheet to your Google Drive.
  • Share the entire sheet via link. Make sure it is View Only - do not allow others to edit.
  • Download the most current version of Charadex and extract the .zip folder.
  • Sign Up or Sign In to Neocities.
  • Go to Edit Site and delete all files except the index.html file.
  • Open the unzipped Charadex file and upload the index.html file and the style folder to your Neocities. Make sure that charadex.css and charadex.js are in the style folder.
  • Go to the index.html file and look for the yourOptions options near the top. Replace the sheetID string with a link to your google sheet.
  • Optional: If you're using an old sheet instead of the new one, fill sheetPage in with the word 'masterlist'. If you changed the masterlist's page name on your sheet, fill sheetPage in with that page name.
  • Your site should be up and running from here.

Now you can finish setting up your sheet or your site.

FAQ

How many characters can this baby hold?

I've tested this with a sheet with about 5000+ entries and it still worked - although load times are slow especially if you're using large images.

If you do have that many entries though, I would really recommend having an actual site and database.

Will you set this up for me?

Unfortunately I don't have the time to even take set-up commissions. If I ever do get the time, I'll announce it on my main Toyhouse!

Where should I host images?

You can host them on your site (but that will require you to give permission to mods to add images). You can also use imgur, discord, toyhou.se, etc, but they do have rules on this type of use and it's best to check their TOS.

Are unwatermarked images safe on here?

No. I've disabled right clicking on images but there's a bunch of ways to get past this.

I broke something!

I'm sorry I can't help a lot on stuff like this. The best I can offer is to just recopy everything.

I've found a bug!

Please do NOT contact me on Toyhou.se. Please file a bug report here and I'll look into it.

Neocities says I need a membership to upload a file!!! You lied!!!!!

You're right. I lied. Take me to jail. Put it on a callout post. But don't remove file and see if the dex still works, that would be blasphemous.
